Description

A console-based Number Guessing Game developed in Java where the user attempts to guess a randomly generated number between 1 and 100. The program provides feedback after each guess and tracks the number of attempts, making it an interactive and beginner-friendly application.

Features

Random number generation within a defined range (1 to 100)

Continuous guessing until the correct number is found

Feedback for each guess indicating whether it is too high or too low

Tracks total number of attempts taken by the user

Handles invalid input using exception handling

Option to restart the game or exit after completion
